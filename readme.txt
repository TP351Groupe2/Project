Pour pouvoir upload des fichiers :

https://windows.github.com/
Telecharger le logiciel.
Au demarage du logiciel identifiez vous avec vos identifiants Github.
Cliquer sur le petit + en haut a gauche du logiciel.
Aller sur clone.
Selectionnez TP351Goupe2.
Selectionnez Project.
Choisisez un emplacement valide sur votre ordinateur.
Validez.

Pour ajouter un fichier vous pouvez travailler directement dans le dossier crée sur votre ordinateur , une fois votre travail terminer vous devrer ajouter un commit avant de pouvoir Sync/publier.
